# 🚗 OBD-II Vehicle Diagnostics System

A real-time vehicle diagnostics system that communicates with a vehicle's OBD-II port using an ELM327 adapter to retrieve diagnostic information and live vehicle parameters.

---

## 📌 Overview

This project enables users to monitor vehicle health by reading data directly from the vehicle's ECU through the OBD-II interface. It provides real-time monitoring of important vehicle parameters and supports diagnostic trouble code (DTC) reading.

---

## ✨ Features

- Read live vehicle parameters
- Display Engine RPM
- Vehicle Speed Monitoring
- Engine Coolant Temperature
- Throttle Position
- Fuel Level Monitoring
- Read Diagnostic Trouble Codes (DTC)
- Clear DTCs (if supported)
- User-friendly interface
- Real-time data updates

---

## 🛠️ Technologies Used

- Python
- OBD Library
- ELM327 OBD-II Adapter

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/obd-ii-vehicle-diagnostics.git
```

Go to project folder

```bash
cd obd-ii-vehicle-diagnostics
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
